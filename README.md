# Smart Privacy Firewall (MVP)

An AI-powered privacy analysis system that automatically analyzes website privacy policies and enforces user-defined privacy rules.

## Features
- Fetches and parses privacy policies
- AI-based clause classification
- Deterministic rule engine
- Risk scoring (SAFE / WARNING / BLOCKED)
- Chrome extension ready backend

## Tech Stack
- Node.js + Express + TypeScript
- PostgreSQL
- FastAPI (Python)
- Chrome Extension (Manifest V3)
